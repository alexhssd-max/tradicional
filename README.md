# Árbol Binario de Búsqueda
**Aprendizaje Automático — Implementación con WEKA**

Visualización interactiva de un Árbol Binario de Búsqueda (BST) con reporte de ordenamientos y búsqueda.

## 🌐 Ver en línea
[GitHub Pages — Árbol Binario](https://TU-USUARIO.github.io/TU-REPOSITORIO)

## 📋 Funcionalidades
- Inserción de nodos con clasificación izquierda/derecha automática
- Visualización gráfica del árbol en tiempo real
- Ordenamientos: **Preorden, Postorden, Inorden (Asc), Inorden (Desc)**
- Contador de pasos por ordenamiento
- Búsqueda con resaltado del nodo encontrado
- Modal con el **código Java completo** (con WEKA)

## 📁 Archivos
| Archivo | Descripción |
|---|---|
| `index.html` | Estructura de la página |
| `style.css` | Estilos y diseño |
| `app.js` | Lógica del árbol binario |
| `Main.java` | Código Java para Apache NetBeans + WEKA |

## ☕ Código Java
El archivo `Main.java` implementa el mismo algoritmo en Java usando la librería WEKA, listo para ejecutarse en **Apache NetBeans**.
